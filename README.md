# 🔢 Random Number Generator

A simple tool to generate random numbers instantly.

## 🚀 Features

* Generates 3 random numbers
* Range: 1 to 100
* Button animation on click

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ▶️ How to Use

1. Click the button
2. Get 3 random numbers instantly

## 📂 How to Run

* Open the HTML file in your browser
